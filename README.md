# Student Performance Analysis (R Project)

This project analyzes student academic performance using R

## Tools Used
R
dplyr
ggplot2

## Analyis Performed
Data manipulation with dplyr
Average score calculation
Gender performance comparison
Data visualization with ggplot2

## Visualizations
The project includes:
Histogram of student scores
Bar chart of student averages
Scatter plot of age vs performance
Box plot comparing gender performance
